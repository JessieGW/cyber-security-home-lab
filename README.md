# cyber-security-home-lab

## Overview
This project documents the design and build of my personal cybersecurity homelab. 
The goal is to create a safe, isolated environment to practice networking, systems administration, and security analysis skills.

This is part of my 90 day cybersecurity portfolio development plan
---
## Objectives
- Build a virtualized lab environment
- Configure isolatd internal networking
- Install and manage multiple operating systems
- Perform basic reconnaissance and vulnerability scanning
- Practice defensive and offensive security tools
- Document all steps for professional portfolio use
---
## Lab Environment
- Host machine with virtualization enabled
- Virtualbox
- Kali Linux (attacker machine)
- Windows 10/11 (target machine)
- Metasploitable (vulnerable machine)
---
## Tools to be Used
- Nmap (network scanning)
- Wireshark (packet analysis)
- Basic Linux terminal commands
- Splunk/Wazuh
--- 
## Lab Goals
By the end of this project, I will be able to 
- Understand basic network architecture
- Identify open ports and services
- Capture and analyze network traffic
- Recognize suspicious activity in logs
- Simulate real-world attack/defense scenarios
