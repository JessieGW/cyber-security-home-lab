# Day 1 - Cybersecurity Home Lab Build

## Objective
Build a controlled virtual lab environment using Virtualbox for cybersecurity practice and network analysis.
---
## Environment Setup
- Virtualization Platform: Virtualbox
- Attacker Machine: Kali Linux
- Target Machine: Windows 10 (installation in progress)
--- 
## Virtual Machine Configuration
- VM Name: Windows-Lab
- RAM: 4GB
- CPU: 2 cores
- Storage 50 GB dynamically allocated
- Installation Type: Custom (clean install)
---
## Installation Status
- Windows ISO downloaded and attached
- VM created successfully
- Windows installation started
---
## Tools to be used in lab
- NMap (network scanning)
- Wireshark (packet analysis)

## Notes
This lab is designed to simulate a basic attacker vs target environment for cybersecurity learning and portfolio development.
